# 证据清单

当视频流程可能要在用户机器上执行时，使用这份清单。

## 来源捕获

- 视频 URL、规范 URL、标题、作者、发布日期、时长。
- 简介文本和所有链接。
- 置顶作者评论、作者回复、最新相关评论、高赞相关评论。
- 评论日期，以及评论是否晚于视频发布。
- 字幕/转写稿、自动字幕、硬字幕、相关弹幕/直播聊天。
- 资源链接、密码、镜像、版本说明、失效或删除的链接。

## 取证访问痕迹

来源访问失败、受限或发生降级时记录：

- 时间、规范 URL、工具、错误类、HTTP 状态或脱敏错误摘要。
- 是否对单条命令使用本地代理；是否使用官方浏览器插件；是否触及登录态或临时 Cookie。
- 如出现人机挑战：记录脱敏挑战文案、`challenge_count`、是否复用当前任务页、挑战前后是否触及登录态、采用了哪些自动替代路径、新增了哪些证据，以及停止原因；不得记录挑战答案或令牌。
- 各层命中项：oEmbed、官方页面结构化数据、info.json、字幕、评论、缩略图、媒体流、公开替代源或无。
- 公开替代源的站点、原视频 ID、作者 ID、时长和抓取时间，以及与官方来源的核对结果。
- 仍缺少的证据，例如媒体流、硬字幕、作者置顶回复、附件列表或精确时间戳。
- 临时凭证或 Cookie 文件的明确路径、授权来源、是否进入 Git，以及任务结束时的清理状态；不得记录凭证值。

## 视觉捕获

- 开始时的整页截图或播放器截图。
- 粗间隔总览拼图。
- 每个 UI 转换、文件/路径/弹窗附近的补充帧。
- 裁剪并放大的字幕区、命令区或路径区。
- 下载前的资源列表截图。
- 修改前的本机路径截图或文件列表。
- 如果 UI 状态重要，执行每一步后的截图。

## 流程还原

每一步都记录：

- 时间戳。
- 来源：视频帧、字幕、作者评论、关联 readme、本机验证。
- 动作。
- 目标路径、应用、账号、网站或设备。
- 所需文件名和来源。
- 预期可见状态或结果。
- 风险：只读、下载、本地状态变化、破坏性、安装/运行二进制、凭据/账号、外部副作用。
- 是否需要确认。
- 执行者：默认 Codex；不可自动化边界单独标记。
- 工具：命令行、脚本、结构化 API、Browser 或 Computer Use。
- 是否占用桌面焦点，以及是否依赖前一步结果。
- 验证方式和停止条件。
- 未知点或路径/版本不匹配。

## 执行分流

- 记录为全自动代理模式，并列出每项工作的自动执行工具。
- 先列出并完成所有不依赖不可自动化边界的 Codex 代理项。
- 插件低可靠时先切换命令行、日志、可访问性接口或其他自动化工具，不转交用户。
- 浏览器操作交给官方 Browser/Chrome 插件；只有桌面应用才交给 Computer Use。
- Computer Use 可能抢焦点前必须告知用户；用户正在使用电脑时不得突然接管。
- 遇到登录输入、验证码求解、支付、UAC/安全桌面或连续失败时，不自动突破边界；切换自动替代路径，无法继续则记录缺口并停止。
- 人机挑战期间暂停页面输入、刷新、下载重试和并发请求；禁止 OCR/坐标脚本自动点击、第三方打码、Cookie 导出或浏览器配置复制。

## 自动执行与缺口记录

每个自动执行批次或阻断点都检查是否写明：

- 开始前的应用、窗口、设备、目标路径和前置状态。
- 按钮或菜单的准确中英文名称、图标或相对位置。
- 单击、双击、右键、输入、选择、拖拽等由工具执行的具体动作。
- 完整输入值、文件名和路径。
- 每一步的等待条件和预期可见结果。
- 不应点击、删除、覆盖或授权的对象。
- 报错、界面不一致或结果异常时的停止条件。
- 自动验证结果；无法观察时记录未验证项、已尝试路径和可恢复起点。

## 执行安全

- 未经用户确认具体文件和目的，不运行第三方二进制文件。
- 未经明确确认，不卸载、不删除、不替换可执行文件、不编辑注册表、不改安全/隐私设置、不改应用数据。
- 官方下载能满足目标时优先官方来源。
- 可行时先备份原文件再替换。
- 不要因为目录名相似就推断它是同一个目标。
- 如果真实机器和视频明显不同，停止并重新判断。

## 最终审计

报告：

- 哪些内容只是读取。
- 下载了什么、路径在哪里、是否保留。
- 修改、创建、删除或备份了什么。
- 打开或保留了哪些标签页、窗口、进程。
- 清理了什么，以及为什么能证明清理安全。
- 哪些内容因为归属或安全性不清楚而故意没有碰。
- 使用了哪种执行模式，Codex、Browser 和 Computer Use 分别完成了什么。
- 是否发生工具或证据路径切换，以及为什么切换或停止。
- 取证降级路径与隐私触达，包括代理、公开替代源、浏览器、登录态和 Cookie；如有临时凭证，说明其清理结果但不输出凭证值。
